# Lab3 Web Server

## 文件说明：

```
Lab3/
├── server.py    (服务器监听)
├── client.py    (用户端发送)
├── index.html   (需要发送的文件)
└── README.md    
```

## 终端命令：

首先在一个终端运行：

```
python server.py --port 8080
```

接着新开启一个终端运行：

```
python client.py --host 127.0.0.1 --port 8080 --file /index.html --clients 50
```
